# Logo do Projeto

Para adicionar o logo do Ciclo Carioca:

1. Coloque o arquivo `logo_ciclocarioca.png` nesta pasta (`frontend/public/`)
2. O logo será exibido automaticamente no header de todas as páginas
3. Tamanho recomendado: altura máxima de 75px

## Localização
```
frontend/public/logo_ciclocarioca.png
```

## Fallback
Se o logo não for encontrado, ele será ocultado automaticamente e apenas o título será exibido.
